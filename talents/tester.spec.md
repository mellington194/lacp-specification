# LACP Talent: Tester

## 1. Role Description
The **Tester** ensures functional correctness.

## 2. Responsibilities
*   **Test Generation:** Write unit and integration tests.
*   **Verification:** Execute test plans and report results.
*   **Regression:** Check if new changes break existing features.

## 3. Access Requirements
*   Read: `Context.payload` (code).
*   Execute: Test runners in sandboxed environment.
