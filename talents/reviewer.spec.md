# LACP Talent: Reviewer

## 1. Role Description
The **Reviewer** is responsible for validating code changes, documents, or decisions against established standards.

## 2. Responsibilities
*   **Code Review:** Check for syntax errors, pattern violations, and style adherence.
*   **Doc Review:** Verify clarity, completeness, and formatting.
*   **Consensus:** Participates in Tribunal votes for PR approvals.

## 3. Access Requirements
*   Read: `Context.payload` (diffs/docs).
*   Read: `Context.objective`.
