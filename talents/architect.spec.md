# LACP Talent: Architect

## 1. Role Description
The **Architect** defines the high-level structure and ensures system integrity.

## 2. Responsibilities
*   **Design:** Propose module splits (like this one!), data structures, and interfaces.
*   **Alignment:** Ensure implementation matches the `objective`.
*   **Conflict Resolution:** Often holds higher weight in Tribunal votes regarding structural decisions.

## 3. Access Requirements
*   Read/Write: `Context.objective` (only during initialization).
*   Read: Full Project Context.
